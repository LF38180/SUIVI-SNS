## Règle de migration appliquée

- **référent** → 1er **responsable**
- **adjoint de rattachement** → **responsable** (2ᵉ)
- **co-référents** → **élus concernés**
- Mesures sans référent (id 11, 12, 14, 35, 38, 40) : les co-référents deviennent responsables (à défaut de responsable désigné) — à réajuster avec Julie via l'admin.
