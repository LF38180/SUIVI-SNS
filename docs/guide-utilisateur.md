# Guide utilisateur — Suivi du programme SNS

## Se connecter

1. Aller sur l'URL de l'application → **Connexion**.
2. Saisir son email (`prenom.nom@mairie-seyssins.fr`) et le mot de passe communiqué.
3. À la première connexion, prévenir l'administrateur pour faire changer le mot de passe temporaire.

## Élu : proposer une mise à jour

1. Onglet **Engagements** → cliquer sur une mesure.
2. Section **Proposer une mise à jour** :
   - Faire glisser le curseur pour fixer le nouvel avancement (0 à 100 %).
   - Ajouter un commentaire (journal de bord) : ce qui a avancé, prochaine étape.
   - Cliquer **Envoyer la proposition**.
3. La proposition part **en attente de validation**. L'avancement officiel ne change pas tant que l'administrateur n'a pas validé.

> Vous pouvez proposer une mise à jour sur **n'importe quelle** mesure (principe de confiance).

## Administrateur : valider ou refuser

1. Le compteur **À valider** en haut indique le nombre de propositions en attente.
2. Onglet **À valider** : pour chaque proposition, vous voyez l'ancienne valeur → la valeur proposée, l'auteur et son commentaire.
3. **Valider** : la valeur proposée devient l'avancement officiel, et une entrée datée est ajoutée à l'historique (inaltérable, alimente la courbe d'évolution).
4. **Refuser** : un motif peut être saisi ; l'avancement officiel reste inchangé.

## Vue publique

La page **/public** est accessible sans connexion. Elle n'affiche que les données **publiées et validées** — jamais les propositions en attente ni le journal de bord interne.

## Exports

- **Imprimer / PDF** : depuis le tableau de bord, bouton *Imprimer / PDF* (à la charte SNS).
- **Export CSV** : bouton *Export CSV* (ouvrable dans Excel).

## Rappels

- Rien n'est public avant validation par l'administrateur.
- Les coûts affichés sont des **catégories indicatives**, jamais des montants budgétaires validés.
